# ⚽ GafferBoard

**GafferBoard** is a high-fidelity tactical soccer whiteboard designed for grassroots and hobbyist coaches. It blends the simplicity of a traditional dry-erase board with the fluid, animated transitions found in modern sports titles like *FC 25*.



## 🌟 Key Features

- **Dynamic Phase Transitions:** Toggle between "Defensive" and "进攻 (Attack)" shapes with a single click. Watch players "spring" into position using professional-grade animation curves.
- **Multi-Size Pitch Support:** Built-in coordinate libraries for **11v11**, **9v9**, and **6v6** team sizes.
- **Tactical Formation Library:** Includes 4-3-3, 4-4-2 (Flat & Diamond), 3-5-2, 3-2-3, and more.
- **Virtual Dry-Erase Kit:** Draw tactical runs, zones, and instructions directly on the pitch with persistent ink that stays visible during transitions.
- **Ball Magnetism:** A smart ball system that allows for free movement but "snaps" to the nearest player when dropped, making it easy to show passing patterns.
- **Static Opponents:** Manually control the "Blue Team" to set up specific defensive blocks or scouting scenarios.

## 🛠️ Tech Stack

* **HTML5 & CSS3:** For the structural layout and pitch aesthetics.
* **Vanilla JavaScript:** Powering the state machine and tactical coordinate maps.
* **CSS Cubic-Bezier Transitions:** To achieve "weighted" player movement.
* **Canvas API:** For the smooth, low-latency drawing engine.

## 🚀 How to Use

1.  **Select Size/Formation:** Use the dropdown to set your team's base layout.
2.  **Toggle Phase:** Click the "Toggle Atk/Def" button to visualize how your team shifts between offensive and defensive shapes.
3.  **Draw Tactics:** Use the Black or Red markers to illustrate movements or passing lanes.
4.  **Manage the Ball:** Drag the ball near any player to have it "stick" to their position.
5.  **Reset:** Use "Reset Players" to return to the starting positions or "Clear Lines" to wipe the board clean.
